# 100 Small Web App Ideas

A hundred little apps you can build just by describing them to an AI like Claude, no coding required. Each one is a single page that runs in a web browser. They are grouped into ten themes, and every idea is tagged by how much work it takes to build.

This pairs well with the "Building with AI" workshop: pick any idea, describe it, and refine it.

## Difficulty legend

- **(easy)**: one screen, works right away, nothing is saved.
- **(medium)**: remembers your information in the browser between visits.
- **(hard)**: saves or shares data with other people (uses a small database).

## How to build any of these

Give Claude the five-part recipe: who it is for, what it should do, a few details, how it should look, and then "build it." Start with an easy one, then change and add things until it feels right.
