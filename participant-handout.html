Today in one minute

You described what you wanted in plain English, and Claude built it. That is the whole trick, and it works the same way every time:

1Describe itSay what you want.
2Claude builds itIt writes the app.
3You tweak itChange, add, repeat.

AI in plain English

  • What it is: an assistant that has read an enormous amount of writing and predicts what should come next, which lets it write things for you.
  • What it is great at: explaining, drafting, summarizing, translating, and building simple web apps.
  • The catch: it can sound completely sure and still be wrong. (People nickname this a "hallucination.") Check anything that matters.
  • Not always current: it may not know very recent news unless it looks it up for you.
  • You are in charge: it does what you ask. Clearer asks get better results.
  • You cannot break it: experiment freely. Start a fresh chat any time.

The prompt recipe

A good ask has five ingredients. Miss one and just add it after.

1Who it is for"for my book club"
2What it should do"track books we have read and our ratings"
3A few details"5 stars, a space for notes"
4How it should look"warm colors, big friendly text"
5Then say build it"now build it as a web app"
Put together: "For my book club, build me a web app that tracks the books we have read with a 5-star rating and a space for notes. Warm colors and big friendly text. Now build it as a web app."

Make it yours: tweak phrases

After Claude builds something, type any of these to change it. Copy them word for word.

  • "Make the text bigger."
  • "Use warmer colors" (or sunset colors, or "my favorite color, teal").
  • "Add a button that says ______."
  • "Add a picture at the top."
  • "Make it work well on my phone."
  • "Add a place for me to type notes."
  • "Put the newest one at the top."
  • "Add a reset button."
  • "Something looks off. Can you fix it?"
  • "Start over, simpler this time."

Three good habits

Save itCopy your work somewhere. Nothing saves automatically.
Check factsIf it matters, verify it. Confident is not the same as correct.
Start overIt is free to try again. A fresh start often beats a long fix.

Mini glossary

AI
Software that can understand and produce language (and images).
Claude
The specific AI assistant we used today.
Prompt
The message you type to ask for something.
Hallucination
When the AI states something wrong as if it were true.
Web app
A small program that runs in a web browser.
Browser
The program you use to visit websites (Chrome, Safari, Edge).
Host / deploy
Putting your app online so others can open it with a link.
Database
Where an app stores information so it is saved or shared.
